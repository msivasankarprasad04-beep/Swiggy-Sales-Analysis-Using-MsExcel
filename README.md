# Swiggy Sales Analysis Dashboard | MS Excel
## Overview

An end-to-end sales analysis dashboard built in Microsoft Excel using Swiggy's food delivery dataset. The project explores sales trends, customer behavior, and geographic performance through interactive visualizations and KPI tracking.

## Key Metrics

###  Metric:                              Value
#### TotalSales:                          ₹53.01M
#### Total Orders:                        1,97,430
#### Average Order Value:                 ₹268.51
#### Average Rating:                      4.34
#### Rating Count:                        55.91L


## Features

Interactive dashboard with slicers (Month, Category, Restaurant)
Monthly & daily sales trend analysis
Quarterly performance breakdown (Q1–Q3)
Sales segmentation by food type (Veg vs Non-Veg)
State-wise and city-wise sales comparison
Top 5 cities by revenue
Weekly sales trend chart

## Tools Used

Microsoft Excel (Pivot Tables, Pivot Charts, Slicers, Conditional Formatting)
